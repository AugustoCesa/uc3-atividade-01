Markdown, CSS e HTML

Markdown

Markdown é um sistema de formatação de texto mais aberto que torna a leitura e a escrita mais simples Com uma codificação mínima, além de facil ele é visualmente mais "limpo", e é facilmente convertido para HTML. E é muito útil por conta de seus arquivos serem pequenos e dificilmente se quebrarem.

CSS

CSS é uma linguagem de formatação para o Html. Cascading Style Sheets é um mecanismo para adicionar estilo a um documento web. O código CSS pode ser aplicado diretamente nas tags ou ficar contido dentro das tags style, ou criar um arquivo com os estilos de CSS.

HTML

HTML abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto" é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores. O HTML não é considerado uma linguagem de programação, já que ele não pode criar funcionalidades dinâmicas. Ao invés disso, com o HTML, os usuários podem criar e estruturar seções, parágrafos e links usando elementos, tags e atributos.
